# T&J Event Services Platform

Internal business management platform for a dry-hire mobile bartending company.

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express
- Prisma ORM
- PostgreSQL

### Features
- Authentication
- Event Management
- POS Drink Tabs
- Stripe Payments
- Package Management

## Project Goals

- Production-style architecture
- Portfolio-quality engineering
- Scalable SaaS foundations
